### 课文内容

#### 1. 观潮（精读课文）

**课文主旨**：描写钱塘江大潮由远而近、奔腾西去的全过程,展现大潮的雄伟壮观。

**重点字词**：
- **易错字**：屹(yì) 昂(áng) 鼎(dǐng) 沸(fèi) 霎(shà) 贯(guàn) 崩(bēng) 震(zhèn)
- **重点词语**：
  - 奇观、农历、据说、笼罩、若隐若现、闷雷滚动、水天相接、横贯江面、漫天卷地、风号浪吼
  - 人声鼎沸：形容人声喧闹，像水在锅里沸腾一样
  - 浩浩荡荡：形容水势盛大的样子
  - 犹如：好像

**重点句式与仿写**：
1. **比喻句**："浪潮越来越近，犹如千万匹白色战马齐头并进，浩浩荡荡地飞奔而来"
   - 仿写：雪花纷纷扬扬地飘落下来，犹如无数洁白的蝴蝶在空中翩翩起舞。

2. **连续动词**："那条白线很快地向我们移来，逐渐拉长，变粗，横贯江面"
   - 仿写：小猫轻轻地走过来，蹲下身子，伸出爪子，抓住了那只老鼠。

**课文结构**：
- 潮来前（第2自然段）：宽阔的钱塘江、平静的江面、人山人海
- 潮来时（第3-4自然段）：从声音和样子两方面描写
  - 声音：隆隆响声→闷雷滚动→山崩地裂
  - 样子：一条白线→拉长变粗→两丈多高的水墙→千万匹白色战马
- 潮去后（第5自然段）：潮头西去、余波涌来、恢复平静

**背诵要求**：背诵第3-4自然段

**典型例题**：
```
1. 按课文内容填空：
午后一点左右，从远处传来（        ）的响声，好像（        ）。
答案：隆隆  闷雷滚动

2. 用"犹如"造句：
参考答案：天上的云彩变化多端，犹如一幅神奇的画卷。

3. 课文是按照什么顺序描写钱塘江大潮的？
答案：潮来前→潮来时→潮去后（时间顺序）
```

#### 2. 走月亮（精读课文）

**课文主旨**：描写阿妈牵着"我"的手在云南洱海畔的月光下散步，抒发浓浓的亲情。

**重点字词**：
- **易错字**：淘(táo) 牵(qiān) 鹅(é) 卵(luǎn) 坑(kēng) 俗(sú) 跃(yuè) 烁(shuò)
- **重点词语**：柔和、鹅卵石、坑坑洼洼、庄稼、风俗、照耀、新鲜、稻穗、闪闪烁烁
- **四字词语**：闪闪烁烁

**重点句式与仿写**：
1. **排比句**："月亮照亮了高高的点苍山，照亮了村头的大青树，也照亮了村间的大道和小路……"
   - 仿写：春风吹绿了山坡，吹绿了田野，也吹绿了校园里的每一个角落……

2. **拟人句**："每个小水塘都抱着一个月亮"
   - 仿写：河边的柳树弯着腰，梳理着自己长长的辫子。

3. **比喻句**："稻田像一块月光镀亮的银毯"
   - 仿写：荷叶像一把把碧绿的大伞，为小鱼撑起一片阴凉。

**课文特点**：
- 反复出现："啊，我和阿妈走月亮"（一咏三叹）
- 多种感官描写：
  - 视觉：细细的溪水、灰白色的鹅卵石
  - 嗅觉：山草和野花的香味、果子的甜香
  - 回忆：阿妈白天洗衣裳、我用树叶做小船

**背诵要求**：背诵第4自然段

**小练笔**：仿照第6自然段，写自己经历过的某个月下情景

**典型例题**：
```
1. 找出文中运用拟人手法的句子并抄写：
答案：每个小水塘都抱着一个月亮。

2. "走月亮"是什么意思？
答案：在有月亮的晚上，到户外月光下游玩、散步、嬉戏。

3. 课文反复出现"啊，我和阿妈走月亮"有什么作用？
答案：表达了"我"和阿妈走月亮时的愉悦心情，体现浓浓的亲情。
```

#### 3*. 现代诗二首（略读课文）

**《秋晚的江上》**：
- 作者：刘大白
- 主要内容：描绘秋江傍晚，余晖浸染的绚丽美景
- 重点诗句："归巢的鸟儿，尽管是倦了，还驮着斜阳回去"
- 想象画面：倦鸟、斜阳、芦苇构成的秋江晚景图

**《花牛歌》**：
- 作者：徐志摩
- 主要内容：描绘花牛在草地上闲暇惬意的生活图景
- 诗歌结构：四节，每节一幅画面
  - 第一节：花牛坐在草地上
  - 第二节：花牛在草地里睡觉
  - 第三节：花牛在草地里散步
  - 第四节：花牛在草地里做梦
- 特点：动静交替（坐→眠→走→做梦）

**重点字词**：巢(cháo) 苇(wěi) 瞬(shùn) 霸(bà) 占(zhàn) 眠(mián)

**典型例题**：
```
1. 《秋晚的江上》描绘了哪些景物？
答案：归巢的鸟儿、斜阳、芦苇、江

2. 《花牛歌》中花牛做了哪些事情？
答案：坐、眠（睡觉）、走、做梦

3. 说说"头白的芦苇，也妆成一瞬的红颜了"的意思。
答案：白色的芦苇被夕阳染成了红色，时间很短暂。
```

#### 4*. 繁星（略读课文）

**课文主旨**：作者巴金回忆三个不同时期、不同地点观赏繁星的情景与感受。

**重点字词**：
- 昧(mèi) 坠(zhuì) 怀(huái)
- 半明半昧、摇摇欲坠、密密麻麻

**课文结构**：
- 从前在家乡庭院：密密麻麻的星星
- 三年前在南京：星群密布的蓝天，光明无处不在
- 如今在海上：半明半昧的星、船在动星也在动

**重点句式**：
- "我好像看见无数萤火虫在我的周围飞舞"（联想）
- "我仿佛看见它们在对我眨眼，我仿佛听见它们在小声说话"（拟人）
- "我觉得自己是一个小孩子，现在睡在母亲的怀里了"（比喻）

**典型例题**：
```
1. 作者在哪些地方看繁星？
答案：家乡的庭院、南京住所的菜园、海上

2. 作者把繁星比作什么？
答案：萤火虫

3. "我觉得自己是一个小孩子，现在睡在母亲的怀里了"表达了作者怎样的感情？
答案：表达了作者对星空的依恋和热爱，感到温馨、安全。
```

### 口语交际：我们与环境

**交际要求**：
1. 围绕话题发表看法，不跑题
2. 判断别人的发言是否与话题相关

**交际话题**：
- 发现问题：身边存在哪些环境问题
- 解决问题：为保护环境提出建议

**典型环境问题**：
- 乱扔垃圾、乱排污水、乱排废气、滥砍滥伐
- 塑料污染、水污染、空气污染、噪声污染

### 习作：推荐一个好地方

**习作要求**：
1. 把推荐的某个地方介绍清楚（在哪里）
2. 把推荐的理由写充分

**推荐角度**：
- 景色角度：这个地方很美
- 风俗角度：可以了解当地的风俗习惯
- 美食角度：有很多好吃的
- 功能角度：可以学习、娱乐等

**习作步骤**：
1. 想清楚要推荐什么地方
2. 从不同角度写推荐理由
3. 把理由写充分、写具体
4. 自读自改，读给同学听

### 语文园地一

#### 交流平台
**学习方法**：边读边想象画面
- 不仅要"看"到样子
- 还要"听"到声音
- "闻"到味道
- 调动多种感官想象

#### 词句段运用

**第一题：形容声音的词语**
- 声音大：锣鼓喧天、震耳欲聋、响彻云霄
- 声音小或无声：鸦雀无声、悄无声息

**第二题：表示时间短暂的词语**
- 忽然、顿时、刹那间、一瞬间、紧接着
- 练习：用这些词语描绘烟花、风、小狗等事物

**典型例题**：
```
1. 读词语，想象画面：
"鸦雀无声"可以想象什么画面？
参考答案：安静的图书馆、考场、冬天的山林等。

2. 用"忽然"写一句话，描述烟花绽放的场景：
参考答案：忽然，一束烟花冲上天空，绽放出五颜六色的光芒。
```

#### 书写提示
- 字的中心要写在横格的中线上
- 保持水平
- 注意字距均匀
- 养成提笔就练字的好习惯

#### 日积月累
**古诗《鹿柴》**（唐·王维）
```
空山不见人，但闻人语响。
返景入深林，复照青苔上。
```
**诗意**：空旷的山中看不见人，只能听到说话的声音。夕阳返照进入深林，又照在青苔上。

**重点字词**：
- 鹿柴(zhài)：栅栏，地名
- 但：只
- 返景：夕阳返照的光
- 复：又

#### 5. 一个豆荚里的五粒豆（精读课文）

**课文主旨**：讲述豆荚成熟裂开后，五粒豆飞到广阔世界中的不同经历，重点描写第五粒豆给病床上的小女孩带来生机和活力。

**重点字词**：
- 豌(wān) 按(àn) 僵(jiāng) 苔(tái) 曾(céng) 框(kuàng) 囚(qiú) 溢(yì) 枪(qiāng) 耐(nài)
- 豌豆、按照、舒适、暖洋洋、曾经、预感、黑暗、囚犯、窗框、耐心、探望、愉快、兴奋、虚弱

**课文结构**：
- 第1-12自然段：五粒豆被射出前各自憧憬未来
- 第13-21自然段：第五粒豆落在窗缝里生根开花，鼓舞小女孩
- 第22-24自然段：其余四粒豆的命运，小女孩恢复健康

**提问策略学习**：
- 既可以针对课文局部提问
- 也可以针对全文提问

**典型问题**：
1. 五粒豆为什么一会儿以为世界是绿色的，一会儿以为世界是黄色的？
2. 小女孩看到豌豆开花后为什么会好起来？
3. 谁才是五粒豆中最了不起的？

**典型例题**：
```
1. 第五粒豆有怎样的经历？
答案：第五粒豆落进顶楼窗子下一个长满青苔的裂缝里，慢慢发芽、开花，带给病床上的小女孩生机和活力。

2. 为什么说小女孩吻豌豆叶的那一天，简直像一个节日？
答案：那一天豌豆开花了，恢复健康的小女孩亲吻了豌豆，一切都是那么美好，作者感到无比高兴。
```

#### 6. 蝙蝠和雷达（精读课文）

**课文主旨**：讲述科学家通过反复研究，揭开了蝙蝠能在夜间飞行的原因，并受此启发给飞机装上雷达。

**重点字词**：
- 蝙(biān) 蝠(fú) 铛(dāng) 蝇(yíng) 证(zhèng) 障(zhàng) 荧(yíng) 屏(píng) 系(jì)
- 雷达、蚊子、即使、科学、横七竖八、证明、研究、驾驶、障碍物、荧光屏、无线电、系统

**课文结构**：
- 第1-2自然段：飞机夜航安全是从蝙蝠身上得到启示
- 第3-7自然段：科学家通过三次实验揭开蝙蝠夜间飞行的秘密
- 第8自然段：揭示飞机安全夜航的原理

**三次实验**：
| 实验次数 | 实验方式 | 实验结果 |
|---------|---------|---------|
| 第一次 | 蒙住眼睛 | 仍能飞行 |
| 第二次 | 塞住耳朵 | 到处乱撞 |
| 第三次 | 封住嘴巴 | 到处乱撞 |

**结论**：蝙蝠靠嘴发出的超声波探路

**提问策略学习**：
- 从内容、写法、启示等不同角度提问

**典型问题**：
1. 蝙蝠和雷达有什么关系？
2. 科学家是怎样揭开蝙蝠飞行的秘密的？
3. 为什么要把蝙蝠比作"没头苍蝇"？

**典型例题**：
```
1. 科学家做了几次实验？每次实验的结果是什么？
答案：三次。第一次蒙住眼睛，仍能飞行；第二次塞住耳朵，到处乱撞；第三次封住嘴巴，到处乱撞。

2. 蝙蝠是怎样在夜间探路的？
答案：蝙蝠一边飞，一边从嘴里发出超声波。超声波向前传播，遇到障碍物就反射回来，传到蝙蝠的耳朵里，蝙蝠就能判断出障碍物的位置。

3. 蝙蝠和雷达有什么相同点？
答案：雷达的天线就像蝙蝠的嘴巴，雷达发出的无线电波就像蝙蝠发出的超声波，雷达的荧光屏就像蝙蝠的耳朵。
```

#### 7. 呼风唤雨的世纪（精读课文）

**课文主旨**：介绍20世纪科学技术的发展历程，展现科学技术改变人类生活的巨大成就。

**重点字词**：
- 唤(huàn) 纪(jì) 技(jì) 赖(lài) 耕(gēng) 潜(qián) 核(hé) 哲(zhé) 任(rèn) 善(shàn) 改(gǎi) 程(chéng) 超(chāo) 亿(yì) 联(lián)
- 呼风唤雨、世纪、技术、程控电话、因特网、依赖、寄托、改观、奇迹、联系

**课文结构**：
- 第1自然段：20世纪是呼风唤雨的世纪
- 第2自然段：科学技术改变人类生活
- 第3-4自然段：对比古今生活的巨大变化
- 第5自然段：科学技术将继续改善生活

**重点诗句**：
"忽如一夜春风来，千树万树梨花开"（唐·岑参）
用来形容20世纪科学技术的飞速发展。

**提问策略学习**：
- 筛选出对理解课文最有帮助的问题

**典型问题**：
1. 为什么说20世纪是"呼风唤雨"的世纪？
2. "发现"和"发明"有什么区别？
3. 现代科学技术怎样改善我们的生活？

**典型例题**：
```
1. 现代科学技术出现之前，人类生活是怎样的？
答案：只能在神话中用"千里眼""顺风耳"和腾云驾雾的神仙，来寄托自己的美好愿望。

2. 现代科学技术给人类生活带来了哪些变化？
答案：登上月球、潜入深海、洞察百亿光年外的天体、探索原子核世界的奥秘；电视、程控电话、因特网使人们的联系更加便捷等。

3. 理解"科学技术是人类创造的"这句话的含义。
答案：科学技术不是自然形成的，而是人类通过不断探索、研究、发明创造出来的。人类利用科学技术改善着自己的生活，创造着美好的未来。
```

### 习作：小小"动物园"

**习作要求**：
1. 抓住家人与动物的相似之处
2. 写出家人的特点

**写作角度**：
- 外貌：胖而憨的爸爸像熊
- 特长：擅长游泳的姐姐像鱼
- 性格：威严的爷爷像大老虎
- 综合：波浪卷、爱吃素、性格温和的妈妈像绵羊

**写作步骤**：
1. 想清楚家人与哪种动物相似
2. 想清楚什么地方相似
3. 把相似之处写清楚、写具体
4. 写出自己生活在"动物园"里的感受

### 单元主题

### 课文内容

#### 9. 古诗三首（精读课文）

**第一首：《暮江吟》（唐·白居易）**
```
一道残阳铺水中，半江瑟瑟半江红。
可怜九月初三夜，露似真珠月似弓。
```
**诗意**：一道残阳倒映在江面上，阳光照射下，江水一半呈现出深深的碧绿，一半呈现出红色。最可爱的是那九月初三之夜，露水像珍珠，新月像弯弓。

**重点字词**：
- 瑟瑟：碧绿色
- 可怜：可爱
- 真珠：珍珠

**第二首：《题西林壁》（宋·苏轼）**
```
横看成岭侧成峰，远近高低各不同。
不识庐山真面目，只缘身在此山中。
```
**诗意**：从正面、侧面看庐山山岭连绵起伏、山峰耸立，从远处、近处、高处、低处看庐山，庐山呈现各种不同的样子。我之所以认不清庐山真正的面目，是因为我自己身在庐山之中。

**哲理**：看问题要全面客观，不能片面。当局者迷，旁观者清。

**重点字词**：
- 题：书写
- 西林：西林寺
- 缘：因为

**第三首：《雪梅》（宋·卢梅坡）**
```
梅雪争春未肯降，骚人阁笔费评章。
梅须逊雪三分白，雪却输梅一段香。
```
**诗意**：梅花和雪花都认为各自占尽了春色，谁也不肯服输，难坏了诗人，难写评判文章。说句公道话，梅花须逊让雪花三分晶莹洁白，雪花却输给梅花一段清香。

**哲理**：人各有所长，也各有所短，要有自知之明。

**重点字词**：
- 降：服输
- 骚人：诗人
- 阁笔：放下笔
- 评章：评议的文章
- 逊：差、不如

**典型例题**：
```
1. 《暮江吟》描写了哪两个时间段的景色？
答案：傍晚（残阳西下）和夜晚（九月初三夜）

2. 《题西林壁》告诉我们什么道理？
答案：看问题要全面客观，不能片面；当局者迷，旁观者清。

3. 《雪梅》中梅花和雪花各有什么特点？
答案：雪花晶莹洁白，梅花清香扑鼻。
```

#### 10. 爬山虎的脚（精读课文）

**课文主旨**：介绍爬山虎叶子和脚的特点及其如何向上攀爬，展现作者连续细致的观察。

**重点字词**：
- 虎(hǔ) 占(zhàn) 铺(pū) 均(jūn) 叠(dié) 空(kòng) 隙(xì) 茎(jīng) 柄(bǐng) 萎(wěi) 瞧(qiáo) 牢(láo) 固(gù)
- 爬山虎、操场、嫩红、舒服、均匀、重叠、空隙、拂过、叶柄、蜗牛、紧贴、休想、牢固

**课文结构**：
- 第1自然段：爬山虎生长的位置
- 第2自然段：叶子的特点和变化
  - 刚长出的叶子：嫩红色
  - 长大的叶子：绿得新鲜，绿得舒服
  - 叶子的排列：均匀，不留一点儿空隙
- 第3-5自然段：爬山虎的脚的位置、样子、颜色
  - 位置：茎上长叶柄的地方反面
  - 样子：像蜗牛的触角，六七根细丝
  - 颜色：嫩红→灰色（巴在墙上的）
- 第6-7自然段：爬山虎如何向上攀爬
  - 步骤：触→巴→拉弯、贴

**准确生动的表达**：
- "一阵风拂过，一墙的叶子就漾起波纹，好看得很"（动态描写）
- "爬山虎的脚触着墙的时候，六七根细丝的头上就变成小圆片，巴住墙"（细节描写）
- "如果……休想……"（假设句，突出爬山虎的脚巴得很牢固）

**观察方法**：
- 长期观察：从刚长出到长大
- 仔细观察：观察到细丝、小圆片等细节
- 按顺序观察：先叶子后脚，先样子后动作

**典型例题**：
```
1. 爬山虎的叶子有什么特点？
答案：刚长出的叶子是嫩红的，长大的叶子绿得新鲜，叶尖一顺儿朝下，在墙上铺得那么均匀，不留一点儿空隙。

2. 爬山虎的脚长在什么地方？是什么样子的？
答案：爬山虎的脚长在茎上长叶柄的地方反面。样子像蜗牛的触角，每根细丝像蜗牛的触角。

3. 爬山虎是怎样向上爬的？
答案：爬山虎的脚触着墙的时候，六七根细丝的头上就变成小圆片，巴住墙。细丝原先是直的，现在弯曲了，把爬山虎的嫩茎拉一把，使它紧贴在墙上。

4. 为什么作者能把爬山虎写得这么生动具体？
答案：因为作者进行了连续细致的观察。
```

#### 11. 蟋蟀的住宅（精读课文）

**课文主旨**：介绍蟋蟀住宅的特点及修建过程，表现蟋蟀的勤劳和才能。

**重点字词**：
- 宅(zhái) 临(lín) 穴(xué) 厅(tīng) 卧(wò) 专(zhuān) 寸(cùn) 址(zhǐ) 良(liáng) 掘(jué) 抛(pāo)
- 住宅、隐蔽、毫不可惜、随遇而安、选择、住址、洞穴、柔弱、搜索、隧道、骤雨、宽敞、简朴

**课文结构**：
- 第1自然段：蟋蟀以住宅闻名
- 第2-6自然段：蟋蟀住宅的特点
  - 选址：排水优良、阳光温和
  - 外部：朝着阳光，很隐蔽，门口收拾得很平坦
  - 内部：九寸深，一指宽，简朴、清洁、干燥、卫生
- 第7-9自然段：蟋蟀如何修建住宅
  - 工具：前足扒土、后足踏地、两排锯
  - 过程：扒土→搬土→踏地→修整→长时间加深加阔

**准确生动的表达**：
- 运用拟人手法："蟋蟀和它们不同，不肯随遇而安"
- 细节描写："当四周很安静的时候，蟋蟀就在这平台上弹琴"
- 对比："别的昆虫大多在临时的隐蔽所藏身"与"蟋蟀慎重地选择住址"

**写作特点**：
- 作者法布尔是法国著名昆虫学家
- 运用拟人手法，将蟋蟀当作人来写
- 观察细致，描写准确

**典型例题**：
```
1. 蟋蟀的住宅有什么特点？
答案：选址排水优良、有温和的阳光；外部朝着阳光、很隐蔽、门口平坦；内部简朴、清洁、干燥、卫生。

2. 蟋蟀是用什么工具修建住宅的？
答案：前足扒土，后足踏地，两排锯修整。

3. 文中哪些地方运用了拟人的手法？
答案："不肯随遇而安""慎重地选择住址""在平台上弹琴"等。

4. 为什么说蟋蟀是"伟大的工程师"？
答案：因为蟋蟀的住宅设计合理，修建需要很长时间，需要耐心和毅力，工具简单却能修建出很好的住宅。
```

#### 12. 盘古开天地（精读课文）

**课文主旨**：讲述盘古开天辟地的神话故事，赞美盘古无私奉献的精神。

**重点字词**：
- 混(hùn) 黑(hēi) 缓(huǎn) 浊(zhuó) 丈(zhàng) 撑(chēng) 竭(jié) 累(lèi) 血(xuè) 液(yè)
- 宇宙、黑暗、上升、下降、逐渐、巨人、倒下、肢体、血液、奔流不息、茂盛

**课文结构**：

- 开天辟地：盘古醒来，劈开混沌，清者上升为天，浊者下降为地

- 化生万物：盘古倒下后，身体各部分化作世间万物

**神话特点**：

- 气魄宏大：盘古身高九万里
- 充满神奇：盘古身体化作万物

**典型例题**：
```
1. 盘古是怎样开天辟地的？
答案：盘古睁开眼睛，用神斧劈向周围的黑暗。轻而清的东西缓缓上升，变成了天；重而浊的东西慢慢下降，变成了地。

2. 盘古倒下后，他的身体发生了哪些变化？
答案：呼出的气息变成了风和云，发出的声音变成了雷声，左眼变成了太阳，右眼变成了月亮，隆起的肌肉变成了山峦，流淌的血液变成了江河，身上的毛发变成了花草树木。

3. 这个神话故事表达了什么？
答案：表达了古代劳动人民对盘古创造天地、无私奉献精神的赞美。
```

#### 13. 精卫填海（精读课文·文言文）

**原文**：
```
炎帝之少女，名曰女娃。女娃游于东海，溺而不复。
故为精卫，常衔西山之木石，以堙于东海。
```

**译文**：
炎帝的小女儿，名叫女娃。女娃到东海去游玩，淹死在东海里，再也没有回来。因此化作精卫鸟，常常衔着西山的树枝和石块，用来填塞东海。

**重点字词**：
- 少女：小女儿
- 溺：淹没
- 复：回来
- 故：因此
- 堙(yīn)：填塞

**学习要求**：
1. 正确、流利地朗读课文
2. 借助注释讲故事，不要求逐字翻译

**典型例题**：
```

2. 精卫鸟在做什么？
答案：精卫鸟常常衔着西山的树枝和石块，用来填塞东海。

3. 精卫填海的故事表现了什么精神？

#### 25. 王戎不取道旁李（精读课文·文言文）

**原文**：
```
王戎七岁，尝与诸小儿游。看道边李树多子折枝，诸儿竞走取之，唯戎不动。
人问之，答曰："树在道边而多子，此必苦李。"取之，信然。
```

**译文**：
王戎七岁时，曾经和许多小孩一起玩耍。看见路边有棵李树，结了很多果实，把树枝都压弯了，许多小孩都争着跑过去摘李子，只有王戎不动。有人问他为什么，他回答说："李树在路边却果实很多，这一定是苦李子。"摘下来一尝，果然如此。

**重点字词**：
- 尝：曾经
- 诸：众多、许多
- 竞走：争着跑过去
- 唯：只有
- 信然：的确这样

**人物品质**：
王戎善于观察、善于思考，能根据事物的特点进行推理判断。

**典型例题**：
```
1. 王戎为什么不去摘李子？

- 要坚持不懈、刻苦练习
- 成功需要付出努力

**典型例题**：
```

2. 纪昌是怎样学习射箭的？
答案：纪昌先练眼力，做到"眼睛一眨不眨"和"把极小的东西看成很大的东西"，然后才开始练习射箭。

3. 从这两个故事中你明白了什么道理？
答案：《扁鹊治病》告诉我们要善于听取正确意见，不要讳疾忌医；《纪昌学射》告诉我们学习要打好基础，坚持不懈。
```

## 阅读理解专项复习

### 一、阅读理解的题型与方法

#### （一）概括主要内容

**方法指导**：
1. **要素串联法**：时间+地点+人物+事情（起因、经过、结果）
2. **段意归纳法**：把每个自然段或每部分的意思连起来
3. **问题归纳法**：课文围绕什么问题讲了哪些内容
4. **标题扩展法**：根据课文标题进行扩展

**典型例题**：
```
例：概括《观潮》的主要内容。
参考答案：课文按照潮来前、潮来时、潮去后的顺序，描写了钱塘江大潮由远而近、奔腾西去的全过程，展现了大潮的雄伟壮观。

方法：运用要素串联法，抓住时间顺序和主要内容进行概括。
```

#### （二）理解词语意思

**方法指导**：
1. **联系上下文理解**：看词语在句子中的意思
2. **拆分理解**：把词语分开理解，再合起来
3. **查字典理解**：借助工具书理解
4. **联系生活实际理解**：结合生活经验理解
5. **找近义词理解**：用熟悉的词语替换

**典型例题**：
```
例1：理解"人声鼎沸"的意思。
方法：拆分理解。"鼎"是古代煮东西的器物，"沸"是水开了。"鼎沸"就是锅里的水开了的样子。"人声鼎沸"形容人声喧闹，像水在锅里沸腾一样。

例2：理解"若隐若现"在句子中的意思："远处的小山，在云雾中若隐若现。"
方法：联系上下文理解。因为有云雾，所以小山一会儿看得见，一会儿看不见。
```

#### （三）理解句子含义

**方法指导**：
1. **抓住重点词语**：理解关键词的意思
2. **联系上下文**：结合前后文理解
3. **体会修辞手法**：理解比喻、拟人等修辞的含义
4. **联系生活实际**：结合自己的经验理解
5. **体会表达效果**：想想为什么这样写

**典型例题**：
```
例1：理解"每个小水塘都抱着一个月亮"的含义。
方法：体会修辞手法。这句话运用了拟人的手法，把水塘映出月亮的倒影，写成水塘抱着月亮，既生动形象，又表达了温馨的情感。

例2：理解"科学技术是人类创造的"这句话。
方法：联系上下文和生活实际。科学技术不是自然形成的，而是人类通过不断探索、研究、发明创造出来的。人类利用科学技术改善着自己的生活，创造着美好的未来。
```

#### （四）体会思想感情

**方法指导**：
1. **抓住重点词句**：从描写中体会情感
2. **关注人物的语言、动作、神态**：从中体会心情
3. **联系时代背景**：理解人物所处的环境
4. **设身处地**：想象自己处在那种情况下的感受

**典型例题**：
```
例：《为中华之崛起而读书》表达了周恩来怎样的思想感情？
参考答案：表达了周恩来少年时期立志振兴中华的远大志向和强烈的爱国情怀。

方法：联系时代背景和课文内容，周恩来看到中国人在租界受欺辱，感到中国太弱了，立志要让中国强大起来。
```

#### （五）分析人物形象

**方法指导**：
1. **抓住人物的语言**：从说的话看性格
2. **抓住人物的动作**：从做的事看品质
3. **抓住人物的神态、心理**：从表情和想法看内心
4. **联系故事情节**：从整个故事看人物特点

**典型例题**：
```
例：西门豹是一个怎样的人？
参考答案：西门豹是一个善于调查研究、聪明机智、为民除害、造福百姓的好官。

方法：联系故事情节分析。西门豹到邺地后先调查了解情况（善于调查研究），然后巧妙破除迷信（聪明机智、为民除害），最后兴修水利（造福百姓）。
```

#### （六）判断修辞手法及作用

**常见修辞手法**：
1. **比喻**：把一种事物比作另一种事物
   - 作用：使表达更生动形象，便于理解
   - 例句："浪潮越来越近，犹如千万匹白色战马齐头并进"

2. **拟人**：把事物当作人来写
   - 作用：使事物更生动、更亲切
   - 例句："每个小水塘都抱着一个月亮"

3. **排比**：结构相似的句子连用
   - 作用：增强语势，使表达更有气势
   - 例句："月亮照亮了高高的点苍山，照亮了村头的大青树，也照亮了村间的大道和小路……"

4. **夸张**：夸大或缩小事物的特征
   - 作用：突出特点，引起联想
   - 例句："飞流直下三千尺"

5. **反问**：用疑问的形式表达肯定的意思
   - 作用：加强语气，引起注意
   - 例句："难道它的眼睛特别敏锐，能在漆黑的夜里看清楚所有的东西吗?"

**典型例题**：
```
例1：判断"浪潮越来越近，犹如千万匹白色战马齐头并进"运用了什么修辞手法，有什么作用？
答案：运用了比喻的修辞手法，把浪潮比作千万匹白色战马，生动形象地写出了浪潮的气势磅礴。

例2："月亮照亮了高高的点苍山，照亮了村头的大青树，也照亮了村间的大道和小路……"运用了什么修辞手法？
答案：运用了排比的修辞手法，增强了语势，更有气势地表现了月光的明亮。
```

### 二、阅读理解练习

**练习一**：
阅读下面的短文，回答问题。

**秋天的雨**

秋天的雨，是一把钥匙。它带着清凉和温柔，轻轻地，轻轻地，趁你没留意，把秋天的大门打开了。

秋天的雨，有一盒五彩缤纷的颜料。你看，它把黄色给了银杏树，黄黄的叶子像一把把小扇子，扇哪扇哪，扇走了夏天的炎热。它把红色给了枫树，红红的枫叶像一枚枚邮票，飘哇飘哇，邮来了秋天的凉爽。

1. 短文把秋天的雨比作什么？
2. 文中运用了什么修辞手法？请举例说明。
3. "扇走了夏天的炎热""邮来了秋天的凉爽"分别是什么意思？

**参考答案**：
1. 短文把秋天的雨比作钥匙和颜料。
2. 运用了比喻和拟人的修辞手法。例如，把秋天的雨比作钥匙，把银杏叶比作小扇子，把枫叶比作邮票；把秋雨当作人，说它"打开秋天的大门"。
3. "扇走了夏天的炎热"指秋天到了，天气凉爽了；"邮来了秋天的凉爽"指秋天带来了凉爽的天气。

**练习二**：
阅读《爬山虎的脚》片段，回答问题。

"爬山虎的脚触着墙的时候，六七根细丝的头上就变成小圆片，巴住墙。细丝原先是直的，现在弯曲了，把爬山虎的嫩茎拉一把，使它紧贴在墙上。爬山虎就是这样一脚一脚地往上爬。"

1. 这段话写了爬山虎向上爬的哪几个步骤？
2. 作者为什么能把爬山虎写得这么清楚？
3. 从这段话中你学到了什么写作方法？

**参考答案**：
1. 写了三个步骤：触墙、巴墙、拉茎。
2. 因为作者进行了连续细致的观察。
3. 学到了按顺序写、抓住细节写、用准确的动词等写作方法。
