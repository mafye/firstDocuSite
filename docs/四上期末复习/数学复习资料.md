## 一、复习与提高

### 1. 加减法的意义

:::tip 📖 加法的意义
一个加数＋另一个加数 = 和
:::

还可以这样理解：
- 被减数 = 差＋减数
- 减数 = 被减数－差

<details>
<summary>📝 例题1</summary>

283＋231 = 514，那么：
- 283 = 514 - 231
- 231 = 514 - 283

**想一想**：加法和减法其实是好朋友，可以互相转换哦！

</details>

### 2. 乘除法的意义

:::tip 📖 乘法的意义
因数×因数 = 积
:::

还可以这样理解：
- 被除数 = 商×除数
- 除数 = 被除数÷商

<details>
<summary>📝 例题2</summary>

求出□中的数：□×23 = 207，□ = 207÷23

**小技巧**：遇到□的题目，想想乘法和除法的关系就能解决！

</details>

### 3. 用两位数乘的竖式笔算方法

:::note 💡 计算步骤
1. 先把相同数位对齐
2. 再从个位乘起，与哪一位上的数相乘，就把积写在那一位上
3. 哪一位上乘得的积满几十，就向前一位进几
4. 最后把乘得的积相加
:::

<details>
<summary>📝 例题3：计算 326×48</summary>

```text
    326
  ×  48
  -----
   2608  (326×8)
  13040  (326×40，注意0要占位！)
  -----
  15648
```

</details>

### 4. 用两位数除的竖式笔算方法

:::note 💡 计算步骤
1. 先看被除数的前两位
2. 如果前两位不够除，就看前三位
3. 除到被除数的哪一位，商就写在那一位上面
4. 每次除后余下的数要比除数小
:::

<details>
<summary>📝 例题4：计算 828÷36 = 23</summary>

```text
    23
  -----
36)828
   72↓
   ---
   108
   108
   ---
     0
```

</details>

## 二、大数的认识

### 1. 一万一万地数

:::tip 📖 计数规律
- 10个一万是十万
- 10个十万是一百万
- 10个一百万是一千万
- 10个一千万是一亿
:::

**计数单位**：一（个）、十、百、千、万……都是计数单位

**数位**：我国的计数习惯，每四个数位是一级——**个级、万级、亿级**

:::note 💡 记忆小窍门
相邻两个计数单位之间的进率都是10，就像爬楼梯，每10个就上一层！
:::

<details>
<summary>📝 例题5</summary>

10个一万是（**十万**），10个十万是（**一百万**）

</details>

### 2. 多位数的读法

:::tip 📖 读数规则
- 先分级，然后从高位读起
- 每一级末尾所有的0都不读
- 其他数位有一个0或连续几个0，都只读一个"零"
:::

<details>
<summary>📝 例题6</summary>

10000001 读作：**一千万零一**

**注意**：中间的连续6个0只读一个"零"哦！

</details>

### 3. 多位数的写法

:::tip 📖 写数规则
- 从高位写起
- 按照数位的顺序写
- 哪一位上一个计数单位的个数为0，就在那一位上写0
:::

<details>
<summary>📝 例题7</summary>

三百二十万零一百 写作：**3200100**

**小提醒**：写数时要特别注意0的位置，不能多也不能少！

</details>

### 4. 亿以内数的比较

:::tip 📖 比较方法
- 位数不同，位数多的数就大
- 位数相同，左起第一位上的数大的那个数就大
- 如果左起第一位上的数相同，就比较左起第二位上的数
- 一直比较到比较出大小为止
:::

<details>
<summary>📝 例题8</summary>

28001020 > 28000120

**为什么？** 前面都一样，比到万位：0 > 0？不对！再往后比，千位：1 > 0，所以第一个数大！

</details>

### 5. 用"亿"、"万"做单位

为了读写方便，把整亿、整万的数改写成用"亿"、"万"做单位的数。

<details>
<summary>📝 例题9</summary>

10010000 改写为：**1001万**

**注意**：这里只是改变了表达方式，数的大小没变哦！

</details>

### 6. 四舍五入法

:::tip 📖 方法说明
求一个数的近似数，主要看省略的最高位上的数：
- 是0、1、2、3、4，就把尾数都舍去
- 是5、6、7、8、9，就把尾数都舍去后向前一位进1
:::

<details>
<summary>📝 例题10</summary>

12345678 约等于（**1235**）万

**解析**：看千位是5，要进位，所以万位的4要进1变成5

</details>

## 三、单位换算公式

### 1. 面积单位：平方米、平方分米、平方厘米

在表示市、区等大的面积时，我们常用**平方千米**来作单位。

:::tip 📊 面积单位换算关系
- 1平方米 = 100平方分米
- 1平方分米 = 100平方厘米
- 1平方厘米 = 100平方毫米
:::

:::note 💡 记忆规律
**高级单位乘以进率等于低级单位，低级单位除以进率等于高级单位**
:::

**生活中的例子**：
- 一张课桌的面积大约是0.5平方米
- 教室的地板面积大约是50平方米
- 一个足球场的面积大约是7000平方米
- 学校操场的面积大约是1公顷（10000平方米）

<details>
<summary>📝 单位换算练习</summary>

- **例题11**：23平方米 = （**2300**）平方分米
- **例题12**：13平方分米 = （**1300**）平方厘米
- **例题13**：24平方厘米 = （**2400**）平方毫米

</details>

### 2. 吨的认识

在计量较重的物品时，通常用比千克大的单位"**吨**"来表示。1吨可以写成1t。

:::tip 📊 换算关系
1吨 = 1000千克
:::

<details>
<summary>📝 例题14</summary>

2吨 = （**2000**）千克

**生活中的例子**：一头大象大约重5吨，一辆小汽车大约重1.5吨

</details>

### 3. 毫升与升的认识

在测量较少的液体的多少时，一般用"**毫升**"作单位，1毫升记作1mL。

在测量较多的液体的多少时，一般用"**升**"做单位，1升记作1L。

:::tip 📊 换算关系
1升 = 1000毫升
:::

<details>
<summary>📝 换算练习</summary>

- **例题14**：4升 = （**4000**）毫升
- **例题15**：5毫升 = （**0.005**）升
- **例题16**：3千克 = （**3000**）克（千克与克的换算）

**小知识**：一瓶矿泉水大约是500毫升，一桶饮水机的水大约是19升

</details>

:::info 📚 扩展知识

体积单位与容积单位的关系：
- 1升 = 1立方分米 = 1000毫升
- 1毫升 = 1立方厘米

**扩展例题**：2立方米 = （**2000**）立方分米 = （**2000**）升

## 四、分数初步认识

### 1. 分数的意义

:::tip 📖 分数定义
把单位"1"平均分成若干份，表示这样的一份或几份的数，叫做分数。
:::

<details>
<summary>📝 例题18</summary>

把一个苹果平均分成5份，每份是这个苹果的 **1/5**

</details>

### 2. 分数比大小

:::tip 📖 比较规律
- 分母相同的分数，分子大的分数就大
- 分子是1的分数，分母越大，分数就越小
- 分母小的分数就越大
:::

<details>
<summary>📝 例题19：比较大小</summary>

- 1/5 < 1/3（分子相同，分母小的大）
- 3/7 < 5/7（分母相同，分子大的大）

**想象一下**：把一个蛋糕平均分成5份和分成3份，分成3份的每一份是不是更大呢？

</details>

### 3. 分数加减计算

:::tip 📖 计算规则
相同分母的分数相加减，**分母不变，分子相加减**
:::

<details>
<summary>📝 例题20</summary>

- 1/5 + 3/5 = 4/5
- 3/7 + 1/7 = 4/7
- 5/8 - 3/8 = 2/8

</details>

### 4. 分数墙的作用

:::note 💡 用途
- 可以比较两个分数的大小
- 可以进行相同分母分数的加减计算
- 可以找出相等的分数
:::

## 五、工作效率、工作时间、工作量

### 1. 工作效率

:::tip 📖 定义
每小时（每分、每天等）完成的工作量叫做工作效率。
:::

### 2. 三者的关系

:::tip 📊 记住这三个公式
- 工作效率 = 工作量÷工作时间
- 工作时间 = 工作量÷工作效率
- 工作量 = 工作效率×工作时间
:::

<details>
<summary>📝 例题21</summary>

小明阅读一本468页的书，18天看完，平均每天看多少页？

**解答**：468÷18 = 26（页）

**答**：平均每天看26页。

**解析**：总页数是工作量，18天是工作时间，每天看的页数是工作效率

</details>

## 六、四则运算

### 1. 什么是四则运算？

:::tip 📖 定义
加法、减法、乘法和除法统称四则运算。
:::

### 2. 没有括号的算式

:::tip 📖 规则
如果只有加、减法或者只有乘、除法，都要从左往右按顺序计算
:::

<details>
<summary>📝 例题22</summary>

- 1＋2＋3＋4＋5 = 15
- 1×2×3 = 6

</details>

### 3. 有乘除和加减的算式

:::tip 📖 规则
要先算乘除，再算加减
:::

<details>
<summary>📝 例题23：2×3－6÷2 = 3</summary>

**计算步骤**：
1. 先算乘法：2×3 = 6
2. 再算除法：6÷2 = 3
3. 最后算减法：6－3 = 3

</details>

### 4. 算式有括号

:::tip 📖 规则
- 要先算括号里面的，再算括号外面的
- 在一个算式里，如果既有圆括号，又有方括号，要先算圆括号里的，再算方括号里的
:::

<details>
<summary>📝 例题24：(1＋2)×2 = 6</summary>

**计算步骤**：
1. 先算括号里：1＋2 = 3
2. 再算乘法：3×2 = 6

</details>

## 七、运算定律

### 一、加法运算定律

#### 1. 加法交换律

:::tip 📖 定律
两个数相加，交换加数的位置，和不变。**a＋b = b＋a**
:::

<details>
<summary>📝 例题25</summary>

165＋93 = 93＋165

</details>

#### 2. 加法结合律

:::tip 📖 定律
三个数相加，可以先把前两个数相加，再加上第三个数；或者先把后两个数相加，再加上第一个数，和不变。**(a＋b)＋c = a＋(b＋c)**
:::

加法的这两个定律在往往结合起来一起使用。

<details>
<summary>📝 例题26</summary>

165＋93＋35 = 93＋(165＋35) = 93＋200 = 293

**小技巧**：看到165和35能凑成200，就先把它们加起来！

</details>

#### 3. 连减的性质

:::tip 📖 定律
一个数连续减去两个数，等于减去那两个数的和。**a－b－c = a－(b＋c)**
:::

<details>
<summary>📝 例题27</summary>

100－23－77 = 100－(23＋77) = 100－100 = 0

</details>

### 二、乘法运算定律

#### 1. 乘法交换律

:::tip 📖 定律
两个数相乘，交换因数的位置，积不变。**a×b = b×a**
:::

#### 2. 乘法结合律

:::tip 📖 定律
三个数相乘，可以先把前两个数相乘，再乘以第三个数；也可以先把后两个数相乘，再乘以第一个数，积不变。**(a×b)×c = a×(b×c)**
:::

<details>
<summary>📝 例题28</summary>

(2×25)×4 = 2×(25×4) = 2×100 = 200

</details>

乘法的这两个定律往往在往结合起来一起使用。

<details>
<summary>📝 例题29</summary>

125×78×8 = (125×8)×78 = 1000×78 = 78000

**小技巧**：看到125和8就要想到凑成1000！

</details>

#### 3. 乘法分配律

:::tip 📖 定律
两个数的和与一个数相乘，可以先把这两个数分别与这两个数相乘，再把积相加。
- **(a＋b)×c = a×c＋b×c**
- **(a－b)×c = a×c－b×c**
:::

**乘法分配律的应用**：

<details>
<summary>📝 类型一：(a＋b)×c = a×c＋b×c；(a－b)×c = a×c－b×c</summary>

**例题30**：(5＋25)×4 = 5×4＋25×4 = 20＋100 = 120

</details>

<details>
<summary>📝 类型二：a×c＋b×c = (a＋b)×c；a×c－b×c = (a－b)×c</summary>

**例题31**：5×4＋25×4 = (5＋25)×4 = 30×4 = 120

</details>

<details>
<summary>📝 类型三：a×99 = a×(99＋1)；a×b－a = a×(b－1)</summary>

**例题32**：2×99 = 2×(99＋1) = 2×100 = 200

</details>

<details>
<summary>📝 类型四：a×99 = a×(100－1) = a×100－a×1；a×102 = a×(100＋2) = a×100＋a×2</summary>

**例题33**：2×99 = 2×(100－1) = 2×100－2×1 = 200－2 = 198

**记忆方法**：看到99就想到100－1，看到101就想到100＋1，这样计算更简单！

</details>

## 八、简便计算

### 1. 连加的简便计算

:::tip 💡 方法
使用加法结合律（把和是整十、整百、整千的结合在一起）
:::

<details>
<summary>📝 例题34</summary>

106－26－74 = 106－(26＋74) = 106－100 = 6

</details>

### 2. 连减的简便计算

<details>
<summary>📝 方法一：连续减去几个数就等于减去这几个数的和</summary>

**例题35**：106－26－74 = 106－(26＋74) = 106－100 = 6

</details>

<details>
<summary>📝 方法二：减去几个数的和就等于连续减去这几个数</summary>

**例题36**：106－(26＋74) = 106－26－74 = 80－74 = 6

</details>

### 3. 加减混合的简便计算

:::tip 💡 方法
算式中的数可以带着符号移动位置，结果不会发生变化
:::

<details>
<summary>📝 例题37</summary>

- 123＋38－23 = 123－23＋38 = 100＋38 = 138
- 146－78＋54 = 146＋54－78 = 200－78 = 122

**小技巧**：先找能凑成整十、整百的数！

</details>

### 4. 连乘的简便计算

:::tip 💡 方法
使用乘法结合律，把常见的数结合在一起
:::

**常见组合**：
- 看见25就找4
- 看见125就找8
- 看见125还可以找80

<details>
<summary>📝 例题38</summary>

200÷25÷4 = 200÷(25×4) = 200÷100 = 2

</details>

### 5. 连除的简便计算

<details>
<summary>📝 方法一：连续除以几个数就等于除以这几个数的积</summary>

**例题39**：200÷25÷4 = 200÷(25×4) = 200÷100 = 2

</details>

<details>
<summary>📝 方法二：除以几个数的积就等于连续除以这几个数</summary>

**例题40**：200÷(25×4) = 200÷25÷4 = 8÷4 = 2

</details>

### 6. 乘、除混合的简便计算

:::tip 💡 规则
第一个数的位置不变，其余的因数、除数可以交换位置（可以先乘，也可以先除）
:::

<details>
<summary>📝 例题41</summary>

27×13÷9 = 27÷9×13 = 3×13 = 39

**小技巧**：看到27和9，想到27÷9=3，先除再乘更简便！

</details>

## 九、圆

### 1. 圆的认识

:::tip 📖 圆的定义
圆上所有的点到固定的点O都有相等的长度r，固定的那一点O叫做**圆心**，r叫做圆的**半径**。
:::

### 2. 直径和半径的关系

:::tip 📊 关系
直径的长度是半径的2倍，即 **d = 2×r**
:::

<details>
<summary>📝 例题42</summary>

一个圆的半径是5厘米，它的直径是（**10**）厘米

</details>

### 3. 用圆规画圆

:::note 💡 步骤
1. 先确定一点作为圆心
2. 把圆规的两只脚分开，使两脚尖之间的长度为r
3. 再把有针尖的一只脚固定在圆心上
4. 把另一只装有铅笔的脚绕圆心旋转一周，就画出了一个半径为r的圆
:::

### 4. 圆是轴对称图形

:::tip 📖 性质
圆是轴对称图形，有无数条对称轴
:::

## 十、直线、线段与射线

### 1. 认识直线、线段与射线

<details>
<summary>📖 直线</summary>

- 可以向两端无限延伸
- 没有端点，不能度量长度
- 读作：直线AB 或 直线BA

</details>

<details>
<summary>📖 线段</summary>

- 不能向两端无限延伸
- 有两个端点
- 读作：线段AB 或 线段BA

</details>

<details>
<summary>📖 射线</summary>

- 可以向一端无限延伸
- 有一个端点，不能度量长度
- 读作：射线AB（A为端点），只有一种读法，从端点读起

</details>

### 2. 画直线

:::tip 📖 规律
- 过一点可画无数条直线
- 过两点只能画一条直线
- 过三点：如果三点在一条线上，经过三点只能画一条直线；如果这三点不在一条线上，那么经过三点不能画出直线
:::

### 3. 两点之间的距离

:::tip 📖 规律
线段比曲线、折线要短
:::

**两点之间的距离**：连接两点的所有线中，**线段最短**

## 十一、角

### 1. 角的静态定义

:::tip 📖 定义
具有公共端点的两条不重合的射线组成的图形叫做角。

这个公共端点叫做**角的顶点**，这两条射线叫做**角的两条边**。
:::

### 2. 角的分类

根据角的大小，我们可以把角分成以下几类：

| 角的类型 | 角度范围 |
|---------|---------|
| **锐角** | 小于90°的角 |
| **直角** | 等于90°的角 |
| **钝角** | 大于90°小于180°的角 |
| **平角** | 等于180°的角 |
| **周角** | 等于360°的角 |

:::tip 📊 角之间的关系
- 1直角 = 90°
- 1平角 = 2直角 = 180°
- 1周角 = 2平角 = 4直角 = 360°
:::

### 3. 典型例题

<details>
<summary>📝 例题42：判断角的类型</summary>

1. 45° 是（**锐角**）
2. 90° 是（**直角**）
3. 135° 是（**钝角**）
4. 180° 是（**平角**）

**解析**：
- 45° < 90°，所以是锐角
- 90°正好是直角
- 135° > 90°且 < 180°，所以是钝角
- 180°正好是平角

</details>

<details>
<summary>📝 例题43：计算</summary>

一个角是60°，它的3倍是多少度？是什么角？

**解答**：60° × 3 = 180°，是平角

</details>

<details>
<summary>📝 例题44</summary>

一个平角减去一个直角，得到的是什么角？

**解答**：180° - 90° = 90°，得到的是直角

</details>

<details>
<summary>📝 例题45：用量角器画一个75°的角</summary>

**步骤**：
1. 先画一条射线，标出顶点
2. 把量角器的中心点对准顶点，0°刻度线对准射线
3. 在75°的刻度处点一个点
4. 从顶点向这个点画另一条射线
5. 标上角的度数

**小提醒**：画角时要注意量角器的0°刻度线要对准射线哦！

</details>
